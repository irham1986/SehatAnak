# SehatAnak
Stunting adalah kondisi gagal tumbuh pada anak balita yang memerlukan deteksi dan pemantauan dini. Aplikasi mobile "SehatAnak" dirancang untuk membantu orang tua memasukkan data pertumbuhan anak (berat, tinggi/panjang, usia) dan menampilkan hasil analisis risiko stunting, serta memberikan rekomendasi dasar.
